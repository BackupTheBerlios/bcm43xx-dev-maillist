Hi,

The problem is you don't have the firmware installed.
You can use as reference:
http://linuxwireless.org/en/users/Drivers/b43#devicefirmware
But I'll try to give you some steps:
 - download: http://downloads.openwrt.org/sources/wl_apsta-3.130.20.0.o
 - download and compile:
http://bu3sch.de/b43/fwcutter/bcm43xx-fwcutter-006.tar.bz2
   or install the package: bcm43xx-fwcutter (if it is available on suse)
 - execute the command: $> sudo bcm43xx-fwcutter -w /lib/firmware
wl_apsta-3.130.20.0.o
 - in order to make things easier for you, reboot (argh!) the system
and try to use the wireless card again.

Regards,

David Cohen
